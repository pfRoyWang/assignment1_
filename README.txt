User Control:
	Player A: uparrow/ downarrow
	Player B: i/k
	
Console Mode(in game):
	"c" to enter the console mode.
	"exit" to exit the console mode.
	"background:<color>" to change the background color.
	"menu" to go back to main menu.
	"gamespeed:<speed(float)>" to change the speed of the game and exit the console mode.
